SpringWebApp
============

A boiler-plate template project for Eclipse that demonstrates a typical Spring-based J2E webapp.

## Development Environment

* Windows 7
* Spring Tool Suite 3.4.0
* JDK 1.7.0_03-b05
* Tomcat 7.0.30
* MySql 5.6.7-rc

## Core APIs
* Spring/MVC 3.2.8
* Spring Security 3.1.4
* Servlet 3.1
* JSP 2.2.1
* Tiles 3.0.1
